# Car Dealership Website
A modern, full-stack web application for managing car dealership operations, built as part of the Hack4Impact-UMD Bootcamp.

## Project Description
This web application provides a comprehensive platform for car dealerships to showcase their inventory and manage customer interactions. 

## Key Features
**User Authentication:** Secure login and signup system using Firebase Authentication \
**Vehicle Inventory:** Browse and search cars by make, model, year, and price \
**Featured Cars:** Dynamic landing page showcasing highlighted vehicles from the database \
**Shopping Cart:** Save up to 2 vehicles with persistent storage across sessions \
**Checkout System:** Multi-step checkout process with shipping and payment forms \
**Order Management:** Complete order history stored in Firestore database \
**Contact System:** Message submission form with data stored in Firebase \
**Responsive Design:** Mobile-friendly interface styled with TailwindCSS
